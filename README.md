# PHPBadBotBlocker
Block Bad Bots using ReCaptcha and MemCache

## Requirements

 `PHP 5.2+`
 `PHP MemCache Library`
 `MemCache Server`

## Captcha Options

We only support Google ReCaptcha in the beta version.
